物件情報取り込みブックマークレットの設定

物件ポータルサイトの詳細ページを開いた状態でボタンを押すだけで、そのページの情報を「物件判定ツール」に自動で取り込めるようにする機能です。一度だけ設定すれば、あとは物件を見るたびにボタンを押すだけで使えます。

ご自身が今見ているページの内容をその場で読み取るだけの仕組みです。サイトを巡回・大量取得するものではありません。

手順1：ブックマークバーを表示する

  • Mac (Chrome/Edge)：Command + Shift + B
  • Windows (Chrome/Edge)：Ctrl + Shift + B

手順2：ボタンをブックマークバーにドラッグする

下のボタンを、表示したブックマークバーまでドラッグ＆ドロップしてください。

📥 物件情報を取り込む

※ クリックしても動作しません。必ず「ドラッグしてブックマークバーに置く」操作をしてください。

ドラッグできない場合（手動で登録）

  1. ブラウザのブックマーク管理画面で「新しいブックマークを追加」を選ぶ
  2. 名前欄に「物件情報を取り込む」と入力
  3. URL（場所）欄に、下のコードを全部コピーして貼り付けて保存

使い方

  1. 不動産ポータルサイトで、確認したい物件の詳細ページを開く
  2. ブックマークバーの「物件情報を取り込む」をクリックする
  3. 物件判定ツールが新しいタブで開き、入力フォームに内容が自動で入る
  4. 内容に誤りがないか確認・修正してから保存する

主要な不動産ポータルサイト（SUUMO・athome・LIFULL HOME'S など）の物件概要表・物件詳細リストに対応した汎用的な抽出方式です。サイトによって項目が取得できない場合は、手動で修正してください。
